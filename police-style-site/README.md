Rwanda Police–Styled Demo
=========================

This is a static demo site designed to resemble the aesthetic and layout patterns of the Rwanda National Police website without copying proprietary content. All text and assets here are original placeholders.

How to view
-----------

Open `index.html` in a browser. If using a local server:

```bash
python3 -m http.server -d /workspace/police-style-site 8000
```

Then visit `http://localhost:8000`.

Notes
-----
- Colors and layout are inspired by public brand cues (navy + gold). No logos or content from the official site are used.
- Replace placeholder images in `assets/img/` with your licensed media.
- Update texts and links to your actual content structure.
