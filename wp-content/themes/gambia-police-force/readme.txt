A professional WordPress theme designed specifically for the Gambia Police Force, based on the Rwanda National Police website design.

== Description ==

The Gambia Police Force theme is a modern, responsive WordPress theme designed to provide a professional online presence for law enforcement agencies. The theme features:

* Responsive design that works on all devices
* Custom post types for news, videos, photos, and services
* Emergency contact numbers prominently displayed
* Social media integration
* Custom color scheme matching police branding
* Easy customization through WordPress Customizer
* SEO optimized
* Fast loading and performance optimized

== Installation ==

1. Upload the theme folder to `/wp-content/themes/` directory
2. Activate the theme through the 'Appearance' menu in WordPress
3. Go to Appearance > Customize to configure theme options
4. Set up your pages and content

== Frequently Asked Questions ==

= How do I customize the theme colors? =

Go to Appearance > Customize > Colors to change the theme colors.

= How do I add emergency contact numbers? =

Go to Appearance > Customize > Contact Information to update emergency numbers and contact details.

= How do I add news articles? =

Use the custom "News" post type in your WordPress admin to add news articles.

= Is the theme mobile responsive? =

Yes, the theme is fully responsive and will work on all devices including mobile phones and tablets.
