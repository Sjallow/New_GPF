# 🎯 Gambia Police Force Theme - Manual Editing Guide

## Overview
Your Gambia Police Force theme is now fully set up for manual editing! You can easily customize all content, images, and information through multiple user-friendly interfaces.

## 🚀 Quick Start

### 1. **Access Your Admin Dashboard**
- Go to: `http://localhost/New_GPF/wp-admin`
- Login with your WordPress credentials

### 2. **Main Editing Interfaces**
You now have **4 different ways** to edit your website:

1. **🎨 GPF Manual Editor** (Recommended for beginners)
2. **⚙️ WordPress Customizer** (Live preview)
3. **🔧 Elementor** (Visual drag-and-drop)
4. **📝 WordPress Admin** (Traditional editing)

---

## 🎨 Method 1: GPF Manual Editor (Easiest)

### Access: WordPress Admin → **GPF Editor**

This is your **main editing interface** with 4 tabs:

#### **Tab 1: Homepage Content**
- **Hero Section Title**: Change the main welcome message
- **Hero Section Description**: Update the description text
- **News Section Title**: Customize "Latest News & Updates"
- **Services Section Title**: Change "Our Services" title

#### **Tab 2: Header Settings**
- **Main Title**: "GAMBIA POLICE FORCE"
- **Slogan**: "Service - Protection - Integrity"
- **Email Address**: Contact email
- **Emergency Number**: 117
- **Phone Number**: Main phone number

#### **Tab 3: Footer Settings**
- **Facebook URL**: Your Facebook page
- **Twitter URL**: Your Twitter account
- **Instagram URL**: Your Instagram profile
- **YouTube URL**: Your YouTube channel

#### **Tab 4: Add Content**
- **Add News Article**: Create news posts
- **Add Service**: Add new services
- **Add Video**: Upload videos
- **Add Photo**: Add photos to gallery
- **Add Page**: Create new pages

## 🔧 Method 3: Elementor (Visual Editor)

### Access: WordPress Admin → **Pages → Edit with Elementor**

#### **For Pages:**
1. Go to **Pages** in WordPress admin
2. Click **"Edit with Elementor"** on any page
3. **Drag and drop** elements to create content
4. **Click "Update"** to save changes

#### **For Posts:**
1. Go to **Posts** in WordPress admin
2. Click **"Edit with Elementor"** on any post
3. Use Elementor widgets to create content

---

## 📝 Method 4: Traditional WordPress Admin

### **Content Management:**

#### **News Articles:**
- Go to **News** in WordPress admin
- Click **"Add New News"**
- Add title, content, and featured image
- Click **"Publish"**

#### **Services:**
- Go to **Services** in WordPress admin
- Click **"Add New Service"**
- Add service details and image
- Click **"Publish"**

#### **Videos:**
- Go to **Videos** in WordPress admin
- Click **"Add New Video"**
- Upload video or add YouTube link
- Click **"Publish"**

#### **Photos:**
- Go to **Photos** in WordPress admin
- Click **"Add New Photo"**
- Upload image and add description
- Click **"Publish"**

---

## 🖼️ Adding Images

### **Step 1: Upload Images**
1. Go to **Media → Library**
2. Click **"Add New"**
3. Upload your images
4. Note the image URLs

### **Step 2: Use Images in Content**
- **Hero Section**: Upload to Media Library, then use in Smart Slider
- **News Images**: Set as featured image when creating news
- **Service Images**: Upload to `/wp-content/themes/gambia-police-force/images/`
- **Logo**: Upload via Customizer → Header Settings

### **Recommended Image Sizes:**
- **Hero Images**: 1200x600px
- **News Images**: 800x400px
- **Service Images**: 400x300px
- **Logo**: 200x200px

---

## 📰 Setting Up News Ticker

### **Using Ditty News Ticker:**
1. Go to **News Ticker** in WordPress admin
2. Click **"Add New Ticker"**
3. Add your news items
4. Copy the shortcode: `[ditty_ticker]`
5. Paste in any page or widget area

### **Auto News Integration:**
- The ticker automatically pulls from your latest news posts
- No manual setup required!

---

## 🎠 Setting Up Hero Slider

### **Using Smart Slider 3:**
1. Go to **Smart Slider** in WordPress admin
2. Click **"Create New Slider"**
3. Add your images and content
4. Set slider ID to "1" (already configured)
5. The slider will appear on your homepage
